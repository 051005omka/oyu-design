# Отчет по производственной практике
## Тема: Разработка веб-сайта для дизайн-студии "OYU Design Group"
**Срок выполнения:** 19 дней

---

### **Неделя 1: Анализ, проектирование и верстка**

**День 1: Анализ предметной области и постановка задачи**
*   Изучение деятельности студии интерьерного дизайна.
*   Анализ сайтов-конкурентов (тренды: минимализм, темные темы, крупная типографика).
*   Формирование Технического Задания (ТЗ): многостраничный сайт, админ-панель, адаптивность.
*   Выбор стека технологий: HTML5, CSS3, JavaScript (ES6+).

**День 2: Настройка окружения и архитектура проекта**
*   Создание структуры папок проекта (assets, styles, scripts).
*   Подключение шрифтов (SF Pro Display) для соответствия бренд-буку.
*   Создание базового файла `index.html` с семантической разметкой (header, main, footer).

**День 3: Разработка дизайн-системы (CSS)**
*   Определение цветовой палитры в `:root` (Deep Green `#003129`, Accent Copper `#86321a`).
*   Настройка глобальных стилей (сброс отступов, настройки box-sizing).
*   Реализация адаптивной типографики с использованием `clamp()`.

**День 4: Верстка шапки (Header) и главной секции (Hero)**
*   Верстка фиксированного меню навигации.
*   Реализация мобильного меню ("бургер-меню") с анимацией открытия.
*   Создание Hero-блока с заголовком и призывом к действию (CTA).

**День 5: Секция "О студии"**
*   Верстка блока с описанием компании.
*   Интеграция фотографии команды.
*   Стилизация текстового блока (выравнивание, межстрочные интервалы).

---

### **Неделя 2: Основной контент и динамика**

**День 6: Разработка каталога стилей**
*   Верстка секции "Стили" с использованием CSS Grid.
*   Создание карточек стилей с эффектом затемнения при наведении.
*   Реализация hover-эффектов (увеличение фото, появление подписей).

**День 7: Программирование логики (JavaScript Basics)**
*   Создание массива данных `stylesList` в `script.js` для динамической генерации контента.
*   Написание функции `renderStyles()` для отрисовки карточек на странице.

**День 8: Страница "Проекты"**
*   Создание файла `projects.html`.
*   Верстка сетки проектов (Portfolio Grid).
*   Реализация JS-структуры `projects` для хранения данных о выполненных работах.

**День 9: Модальные окна (Popups)**
*   Разработка HTML-каркаса модального окна для просмотра деталей проекта.
*   Стилизация модального окна (Backdrop-filter, анимация появления).
*   Реализация логики открытия/закрытия окон через JS.

**День 10: Слайдер изображений**
*   Разработка функционала галереи внутри модального окна.
*   Программирование кнопок "Вперед/Назад" для переключения слайдов.
*   Обработка краевых случаев (цикличность слайдера).

---

### **Неделя 3: Админ-панель и интерактивность**

**День 11: Проектирование Админ-панели**
*   Создание `admin.html`.
*   Верстка формы входа (Login Form).
*   Защита входа простейшей авторизацией (проверка пароля в JS).

**День 12: Функционал управления контентом (CMS)**
*   Реализация форм добавления новых проектов и стилей.
*   Работа с `localStorage` для сохранения данных без использования серверной БД.
*   Реализация функций `addProject()` и `deleteProject()`.

**День 13: Редизайн Админ-панели**
*   Полная стилизация панели под основной сайт (`admin.css`).
*   Внедрение Glassmorphism (эффект матового стекла) для блоков управления.
*   Стилизация полей ввода и кнопок действий.

**День 14: Анимации и визуальные эффекты**
*   Подключение `IntersectionObserver` для анимации появления элементов при скролле (Scroll Reveal).
*   Добавление плавных переходов страниц (`page-fade-in`).
*   Улучшение микровзаимодействий (кнопки, ссылки).

**День 15: Продвинутая стилизация (Visual Polish)**
*   Реализация эффекта размытия "до колен" для фото в секции "О студии" (CSS Masking / Backdrop-filter).
*   Добавление акцентных рамок и теней.
*   Тонкая настройка градиентов.

---

### **Неделя 4: Интеграции и тестирование**

**День 16: Форма обратной связи**
*   Верстка формы контактов в футере.
*   Валидация полей ввода (HTML5 attributes).
*   Стилизация состояний фокуса и ошибки.

**День 17: Интеграция с Telegram API**
*   Регистрация Telegram-бота (@BotFather).
*   Написание асинхронной функции (`fetch`) для отправки заявок с сайта в Telegram-чат администратора.
*   Обработка ответов сервера и вывод уведомлений пользователю.

**День 18: Оптимизация и рефакторинг**
*   Оптимизация производительности CSS (удаление глобальных `transition`).
*   Проверка кроссбраузерности (Chrome, Firefox, Safari).
*   Тестирование мобильной версии и исправление багов верстки.

**День 19: Финальная сдача проекта**
*   Подготовка документации.
*   Финальная проверка всех путей и ссылок.
*   Демонстрация работы сайта заказчику (руководителю практики).
